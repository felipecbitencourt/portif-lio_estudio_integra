# Pilar 2 - Tecnologias

## Proposito do pilar

O pilar de tecnologias sustenta o desenvolvimento tecnico dos materiais educacionais interativos com foco em compatibilidade, desempenho e manutencao evolutiva. A proposta e garantir que os cursos funcionem de forma confiavel em diferentes LMS, navegadores, sistemas operacionais e dispositivos.

## Como aplicamos na pratica

### Desenvolvimento SCORM

Desenvolvemos cursos empacotados em SCORM (com manifestacao padrao e estrutura de publicacao), garantindo rastreabilidade de progresso, status e conclusao no LMS do cliente. Priorizamos implementacoes robustas para ambientes corporativos heterogeneos.

### Front-end com HTML, CSS e JavaScript

Construimos interfaces educacionais com base web padrao para maximizar compatibilidade. HTML semantico, CSS responsivo e JavaScript modular permitem controlar navegacao, interatividade, avaliacao e feedback com previsibilidade tecnica.

### React quando o projeto exige maior dinamismo

Em projetos com maior complexidade de estado, componentes e reuso de interface, utilizamos React para organizar a aplicacao de forma escalavel. A decisao de usar framework e orientada por necessidade tecnica, sem comprometer entrega em SCORM.

### Python em etapas de apoio e automacao

Utilizamos Python em rotinas de apoio ao desenvolvimento, como processamento de dados, automacao de tarefas repetitivas, validacoes de conteudo e geracao de artefatos tecnicos quando aplicavel.

### Compatibilidade entre sistemas e dispositivos

Projetamos para funcionamento consistente em diferentes combinacoes de:

- Windows, macOS, Android e iOS;
- navegadores modernos (Chrome, Edge, Firefox, Safari);
- desktop, tablet e smartphone.

A estrategia tecnica inclui testes de comportamento responsivo, navegacao por touch e fallback para variacoes de ambiente corporativo.
